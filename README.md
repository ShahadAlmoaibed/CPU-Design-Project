# CPU Design Project

## Course
Computer Organization

## Description
Designed and implemented a single-cycle CPU and a pipelined 16-bit MIPS-like CPU using Logisim-Evolution as part of a Computer Organization course project. The project involved designing the processor datapath, implementing the control logic, supporting a custom instruction set architecture (ISA), and verifying CPU functionality through simulation.

## Project Type
Team project (2 students)

## Technologies
- Logisim-Evolution
- Digital Logic Design

## Files
- Phase 1 implementation
- Phase 2 implementation

## How to Open
1. Open the project using Logisim-Evolution.
2. Load either the Phase 1 or Phase 2 .circ file.
3. Run the simulation to test the CPU design.

## Learning Outcomes
- CPU datapath design
- Control logic implementation
- Instruction Set Architecture (ISA) design
- CPU verification through simulation
